# pet-cloud-adoption

## 项目介绍（Python）
流浪猫/狗云收养系统，旨在解决流浪猫/狗救助站的建设需求与经济压力的不平衡，以及爱心人士对流浪猫/狗极高关注度与力所能及范围小的矛盾。

爱心人士上云收养流浪猫/狗，缓解救助站经济压力，让更专业的人照顾它们，可多对一，为无能力赡养的人创建可奉献爱心的平台。

救助站负责记录流浪猫/狗的点滴，为领养者提供情绪价值。

## 环境配置
1.  **基础环境**
    - Python 版本：Python 3.6 及以上版本
    - GUI 框架：tkinter（Python 标准库，随 Python 一同安装，无需额外安装）

2.  **数据库依赖**
    - 数据库：MySQL 5.7 / 8.0
    - Python 数据库驱动：pymysql（第三方库，需要手动安装）

## 功能说明
前台分为求助站端和爱心人士端，仅围绕该想法完成数据库设计与注册与登录窗体的创作：

1.  **用户角色**：求助站和爱心人士分别对应不同的窗体
2.  **注册**：根据用户角色可以选择注册求助站和爱心人士
3.  **登录**：根据用户角色可以登录之后打开不同的窗体
4.  **退出**：关闭当前窗体，退出应用程序
